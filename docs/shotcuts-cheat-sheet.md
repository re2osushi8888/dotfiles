# ショートカットチートシート
シェル・nvim・weztermのショートカットを被らないように設定する

## zshショートカット
基本ctrlとaltを使う
bindkeyは別のターミナルに入るときのことを考えて、デフォルトのemacsモードを使うようにする

### カーソル移動
- `Ctrl + A`：行頭へ
- `Ctrl + E`：行末へ
- `Alt + B`：1単語左へ
- `Alt + F`：1単語右へ
- `Ctrl + F`：1文字右へ（→と同等）
- `Ctrl + B`：1文字左へ（←と同等）
- `Ctrl + XX`：カーソル位置 ↔ 行頭 をトグル

### 削除・編集
- `Ctrl + H` / `Backspace`：1文字削除（左）
- `Ctrl + D`：1文字削除（右） / 何もなければ EOF
- `Ctrl + W`：カーソルから左の1単語を削除
- `Alt + D`：カーソルから右の1単語を削除
- `Ctrl + U`：カーソルより左を全部削除
- `Ctrl + K`：カーソルより右を全部削除
- `Ctrl + Y`：直近に削除（kill）した内容を貼り付け（yank）
- `Alt + T`：直前2単語を入れ替え
- `Alt + U`：右側の単語を大文字化
- `Alt + L`：右側の単語を小文字化

### 履歴（検索・展開）
- `!!`：直前のコマンドを再実行
- `!$`：直前コマンドの「最後の引数」だけ展開
- `Alt + .`: 直前のコマンドの「最後の引数」だけを展開
- `!*`：直前コマンドの「全引数」展開
- `!string`：最近の `string` で始まるコマンドを実行
- `!?string?`：`string` を含む最近のコマンドを実行

### 補完（zsh）
- `Tab`：補完
- `Tab` 連打：候補一覧/絞り込み（設定次第）
- `Ctrl + I`：Tab と同義
- `Ctrl + N` / `Ctrl + P`：補完候補や履歴を上下（設定次第）
- `Alt + .`：直前コマンドの最後の引数を挿入（挿入版 `!$`）

### 画面・ジョブ制御
- `Ctrl + L`：画面クリア（`clear` 相当）
- `Ctrl + C`：実行中断 / 入力行を破棄
- `Ctrl + Z`：一時停止（suspend）→ `fg`/`bg` で復帰
- `Ctrl + D`：EOF（シェル終了になることも）

### fzfのショートカット
- `Ctrl + R`：コマンド実行履歴から検索し、選択して再入力
- `Ctrl + T`：カレントディレクトリ内のファイル・フォルダを検索し、パスを挿入
- `Alt + C`：サブディレクトリを検索し、そのディレクトリへ cd する

## weztermショートカット

> **NOTE**: 現在は必要最小限のキーバインディングのみを定義しています。今後、必要な機能が出てきたら随時追加していく方針です。
> 追加する際は、keybinds.lua (`config/.config/wezterm/keybinds.lua`) も合わせて更新してください。

### Leader Key
- **Leader Key**: `Ctrl + Q`（1秒タイムアウト）
  - Leader keyを押した後、1秒以内に次のキーを押すことで各機能を実行
  - tmux風の操作感

### 基本操作
- `Shift + Ctrl + T`：新しいタブを開く
- `Shift + Ctrl + W`：タブを閉じる（確認あり）
- `Ctrl + Tab`：次のタブへ移動
- `Shift + Ctrl + Tab`：前のタブへ移動
- `Shift + Ctrl + 1-9`：特定のタブに移動（9は最後のタブ）
- `Shift + Ctrl + C`：コピー
- `Shift + Ctrl + V`：ペースト
- `Shift + Ctrl + P`：コマンドパレット
- `Alt + Enter`：フルスクリーン切替

### スクロール
- `Shift + PageUp`：1ページ上にスクロール
- `Shift + PageDown`：1ページ下にスクロール

### フォントサイズ
- `Ctrl + +` / `Ctrl + =`：フォントサイズを拡大
- `Ctrl + -`：フォントサイズを縮小
- `Ctrl + 0`：フォントサイズをリセット

### 透過率調整
- `Ctrl + Alt + =`：透過率を上げる（+0.1）
- `Ctrl + Alt + -`：透過率を下げる（-0.1）

### 背景ぼかし（Windows Acrylic）
- `Shift + Ctrl + O`：ぼかしのオン/オフ切り替え
- `Ctrl + Alt + O`：ぼかしのオン/オフ切り替え（代替）

### ペイン操作（Leader Key経由）
- `Leader + |`：ペインを左右に分割（縦線で分割）
- `Leader + -`：ペインを上下に分割（横線で分割）
- `Leader + x`：ペインを閉じる（確認あり）
- `Leader + z`：ペインズーム切替
- `Leader + h`：左のペインへ移動
- `Leader + j`：下のペインへ移動
- `Leader + k`：上のペインへ移動
- `Leader + l`：右のペインへ移動
- `Leader + p`：ペイン選択モード（各ペインにラベル表示）

### ペインリサイズモード（Leader Key経由）
- `Leader + s`：リサイズモード起動
  - `h`：左に拡大
  - `j`：下に拡大
  - `k`：上に拡大
  - `l`：右に拡大
  - `z`：ペインを最大化/元に戻す（トグル）
  - `Enter` / `Escape`：リサイズモード終了

### Workspace管理（Leader Key経由）
- `Leader + w`：Workspace一覧を表示
- `Leader + W`（Shift）：新規Workspaceを作成
- `Leader + $`：Workspace名を変更

### コピーモード（Vim操作）
- `Leader + c`：コピーモード起動
  - **移動**: `h/j/k/l`（Vim風カーソル移動）
  - **単語移動**: `w`（次の単語）、`b`（前の単語）、`e`（単語末尾）
  - **行内移動**: `0`（行頭）、`$`（行末）、`^`（行頭の非空白）
  - **ページ移動**: `Ctrl + F`（次ページ）、`Ctrl + B`（前ページ）、`Ctrl + D`（半ページ下）、`Ctrl + U`（半ページ上）
  - **スクロール**: `g`（スクロールバック先頭）、`G`（スクロールバック末尾）
  - **選択**: `v`（文字選択）、`V`（行選択）、`Ctrl + V`（矩形選択）
  - **検索**: `/`（検索モード起動）
  - **コピー**: `y`（コピーして終了）
  - **終了**: `q` / `Escape` / `Ctrl + C`

### 検索モード（コピーモード内で `/` から起動）
- `/`：検索モード起動（大文字小文字を区別）
- 文字入力：検索パターン入力
- `Enter`：検索を確定してコピーモードに戻る
- `↑`：前のマッチへ
- `↓`：次のマッチへ
- `Ctrl + N`：次のマッチへ
- `Ctrl + P`：前のマッチへ
- `Ctrl + R`：マッチタイプ切り替え（文字列 / 大小無視 / 正規表現）
- `Ctrl + U`：パターンをクリア
- `Escape`：検索モード終了（コピーモードも閉じる）

### ショートカットの競合チェック
**zshとの競合なし:**
- weztermのLeader Key（`Ctrl + Q`）はzshで基本的に使用されていない
- Leader Key経由のショートカットはタイムアウト内でのみ有効なので、zshと競合しない
- 基本操作のCtrlキーは主にweztermが優先的にキャプチャするため、zshには届かない

**Vimとの競合回避:**
- Vimで重要な`Ctrl+C/V/P/T/W/0`は使用せず、`Shift+Ctrl`組み合わせを使用
- Leader Key（`Ctrl+Q`）はVimでも基本的に使われていない
- これにより、Vim内で標準的なキーバインディングが正常に動作する
